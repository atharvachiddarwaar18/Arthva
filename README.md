# Arthva

**Clarity in Every Expense**

Arthva is a modern, minimal expense tracking app designed for group trips and personal use. It helps users track shared expenses, split costs intelligently, and visualize spending through a clean, premium dashboard.

---

## 🚀 Features

### 👥 Trip Expense Tracker (Core)

* Create trips and add members
* Add and split expenses (equal/custom)
* Real-time balance tracking (who owes whom)
* Smart settlement system (minimized transactions)
* Group dashboard with spending analytics
* Budget tracking with visual indicators
* Auto capture of date, time, and location

### 🧍 Individual Expense Tracker

* Track personal expenses
* Categorize spending (Food, Travel, Misc)
* View total expenditure

---

## 📊 Dashboard Highlights

* Group-level spending visualization
* Budget vs actual tracking
* Member-wise balance (paid vs owed)
* Clean, modern graphs and insights

---

## 🛠️ Tech Stack

* React Native (Expo)
* Supabase (Backend & Realtime DB)
* PostgreSQL
